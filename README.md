### Run Locally

To run **Clone Website-Filmlane** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/riddhikapil30/Clone-Website.git
```

Windows:

```bash
git clone https://github.com/riddhikapil30/Clone-Website.git
```
